# Lichess Offline Comments Fork

## Overview
This fork transforms the Lichess mobile app into a completely offline chess application with mandatory move comments.

## Key Features
- Offline-only chess (no internet required)
- Mandatory move comments (30+ chars when enabled)
- Two-phase move system
- Unlimited and Correspondence time controls only
- Local profile with nickname/avatar
- SQLite storage for games and comments
- PGN export with comments
- CI/CD pipeline for automated builds
